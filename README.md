Описание программы

Программа предназначена для обработки даты рождения пользователя и выполнения нескольких связанных задач: вычисления дня недели, определения високосного года, подсчёта возраста и отображения даты в виде «звёздочного» табло, похожего на электронные цифры.

Основные функции программы
1. Ввод даты рождения

Пользователь последовательно вводит:

день (1–31),

месяц (1–12),

год (например, 1990).

Программа проверяет корректность введённой даты:

существуют ли такие день, месяц и год,

корректен ли день для выбранного месяца.

Если дата неверная — программа выводит сообщение об ошибке.

2. Определение дня недели

Функция day_of_week() вычисляет, на какой день недели приходилась введённая дата рождения.

Дни недели выводятся на русском языке:

понедельник

вторник

среда

четверг

пятница

суббота

воскресенье

3. Определение високосного года

Функция is_leap_year() проверяет:

делится ли год на 4,

не делится ли на 100,

либо делится на 400.

Результат:
«да» — если год високосный
«нет» — если нет

4. Расчёт возраста пользователя

Функция calculate_age() определяет количество полных лет, сравнивая дату рождения пользователя с текущей датой, которая задана в программе как:

23 ноября 2025 года

Если день рождения в текущем году еще не наступил — возраст уменьшается на 1 год.

5. «Звёздочный» вывод даты

Программа отображает дату рождения в формате:

дд мм гггг


Каждая цифра рисуется блоком из звёздочек *, имитируя электронный дисплей.

Пример (для цифры «2»):

 *** 
*   *
    *
   *
  *
 *   
*****


Аналогично выводятся все цифры даты.

Структура программы

Программа состоит из следующих частей:

Набор шаблонов цифр (словарь DIGITS)
Описание каждой цифры в виде 7 строк символов.

Справочник дней недели на русском языке

Функции:

is_leap_year(year)

day_of_week(day, month, year)

calculate_age(day, month, year)

render_star_number_string(s)

read_int(prompt, min, max) — безопасное считывание числа

valid_date(day, month, year) — проверка корректности даты

Главная функция main()
Выполняет весь сценарий: ввод → вычисления → вывод результата → звёздочный вывод.

Итог

Программа:

корректно обрабатывает дату рождения пользователя,

определяет день недели и высокосность года,

рассчитывает возраст на заданную текущую дату,

красиво выводит дату в виде «звёздочного» табло.
